# Bedrock-Dedicated-Server-Tool
I've worked on this on-and-off for a couple days now as I started to host a server to play with some friends.
Installing mods was a bit of a pain in the arse so I decided to try and automate it.
This is the result of that.

Programmed in Python.

### [Installation]
 - Navigate to [Install Requirements](https://github.com/carpetbomb/Bedrock-Dedicated-Server-Tool/releases/tag/Requirements)
 - Then install the [latest release](https://github.com/carpetbomb/Bedrock-Dedicated-Server-Tool/releases/latest)
